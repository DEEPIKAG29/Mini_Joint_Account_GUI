# Mini_Joint_Account_GUI
A simple Python mini-project demonstrating object-oriented programming concepts and operator overloading with a graphical interface using Tkinter. Users can enter two account holders’ names and balances, and the program combines them into a joint account, displaying the result dynamically in the GUI.

## Features
-BankAccount class with operator overloading (__add__) to merge accounts.
-Interactive Tkinter GUI for user input.
-Handles invalid numeric input.
-Displays joint account holders and total balance.
-Clean interface with heading and thank-you message.

## Usage
-Run the Python file Mini_Joint_Account.py.
-Enter the names and amounts for both account holders.
-Click Submit to see the joint account and total balance.

## Technologies
-Python 3
-Tkinter (GUI)
